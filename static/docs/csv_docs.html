Introduction

This document provides information for all of the voteview CSV files.

Citation: Lewis, Jeffrey B., Keith Poole, Howard Rosenthal, Adam Boche, Aaron Rudkin, and Luke Sonnet (2017). Voteview: Congressional Roll-Call Votes Database. https://voteview.com/

DW-NOMINATE scores are estimated with fixed dimension weights of w = (1.0, 0.4158127) and uncertainty parameter β = 7.7793487.

Data

Member

congress

Integer 1+. Number of the Congress in which the member served. e.g. 115, for the 115th Congress.

chamber

House, Senate, or President. The chamber in which the member served.

icpsr

Integer 1-99999. Identifier for the member. In general, each member receives a single ICPSR identifier applicable to their entire career. A small number of members have receieved more than one; this can occur for members who have switched parties and members who became president. This allows a new NOMINATE estimate to be produced for separate appearances of a member in different roles.

state_icpsr

Integer 0-99. Identifier for the state represented by the member.

district_code

Integer 0-99. Identifier for the district that the member represents within their state. Senate members are given district_code 0. At-large members have been assigned 99, 98, and 1 in various circumstances.

state_abbrev

String. Postal abbreviation for state , e.g. AL for Alabama.

party_code

See Party documentation: party_code.

occupancy

Integer 1+. ICPSR occupancy code. ICPSR Occupancy Code -- 0=only occupant; 1=1st occupant; 2=2nd occupant; etc. Note: This item is considered legacy information and not been verified.

last_means

Integer 1-5. ICPSR Attain-Office Code. Indicator for the member's last means of attaining office. Note: Not maintained or verified.

code description
1 general election
2 special election
3 elected by state legislature
5 appointed

bioname

String. Name of the member. Surname first. For most members, agrees with the Biographical Directory of Congress.

bioguide_id

String. Member identifier in the Biographical Directory of Congress.

born

Integer. Year of member's birth.

died

Integer. Year of member's death.

dim1

NOMINATE dimension 1 estimate.

dim2

NOMINATE dimension 2 estimate.

log_likelihood

NOMINATE log-likelihood of estimate.

geo_mean_probability

NOMINATE geometric mean probability of estimate.

number_of_votes

Number of votes cast by the member during given congress.

number_of_errors

Number of NOMINATE prediction errors for member during given congress.

conditional

Integer 0-1. 1 indicates NOMINATE was estimated conditionally for given member. 0 otherwise.

nokken_poole_dim1

Nokken-Poole estimate first dimension. (Nokken, Timothy P. and Keith T. Poole. 2004. "Congressional Party Defection in American History." Legislative Studies Quarterly, 29:545-568, 2004)

nokken_poole_dim2

Nokken-Poole second dimension. See nokken_poole_dim1.

Rollcall

congress

Integer 1+. Number of the Congress in which the rollcall occurred. e.g. 115, for the 115th Congress.

chamber

House or Senate. Chamber in which the rollcall occurred.

rollnumber

Integer 1+. Counter for each rollcall. Starts from 1 in the first rollcall of each congress. Excludes quorum calls and vacated votes. For these reasons it does not match the clerk_rollnumber.

date

Date on which the rollcall took place. Formatted YYYY-MM-DD.

session

1 or 2. Indicator for the of congress in which a rollcall occurred.

clerk_rollnumber

Number assigned by Congress to the rollcall. Starts from 1 at the beginning of each session (not at the beginning of each congress). Compare to rollnumber.

mid_1

NOMINATE first-dimension midpoint estimate.

mid_2

NOMINATE second-dimension midpoint estimate.

spread_1

NOMINATE first-dimension spread estimate.

spread_2

NOMINATE second-dimension spread estimate.

log_likelihood

NOMINATE estimate log-likelihood.

bill_number

String. Number of the bill on which a rollcall was held.

vote_result

String. Official result of rollcall. May not exist for all votes.

vote_desc

Description of the rollcall matter, assigned by Congressional staff.

vote_question

String. Question addressed by the rollcall. e.g. "On Agreeing to the Amendment".

dtl_desc

Description of the rollcall matter, collected from historical sources.

Vote

Denotes a vote cast by a member on a rollcall.

congress

See Rollcall documentation: congress.

chamber

See Rollcall documentation: chamber.

rollnumber

See Rollcall documentation: rollnumber.

icpsr

See Member documentation: icpsr.

cast_code

Integer 0-9. Indicator of the vote cast by the member on the rollcall.

cast_code description
0 Not a member of the chamber when the rollcall vote was taken
1 Yea
2 Paired Yea
3 Announced Yea
4 Announced Nay
5 Paired Nay
6 Nay
7 Present (some Congresses, also not used some Congresses)
8 Present (some Congresses, also not used some Congresses)
9 Not Voting

prob

NOMINATE estimated probability of the recorded vote.

party_name

Name of party.

n_members

Number of party members during congress.

dim1_median

dim2_median

dim1_mean
